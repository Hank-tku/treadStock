# 股势 TrendStock - 产品规划

## 概述
TrendStock 是一个股票分析工具和策略分析容器。通过 API 参数生成策略，并通过周期性策略打分和自行迭代来持续优化。

### 核心理念
- **易用性**：通过自然语言生成策略 - 结合 API 生成基础策略配置，通过自然语言生成策略算法
- **自行迭代**：根据市场实际情况，周期性复盘，从各个维度（日、周、月等）给策略打分，给出迭代方向和分析结果，生成 Check List 提供给用户，用户确认后更新策略算法

---

## 数据源
- **API**: AkShare (https://akshare.akfamily.xyz/data/stock/stock.html)
- 当前使用 East Money API 作为行情数据源，后续切换到 AkShare
- 新闻数据：财联社、第一财经等

## 核心数据视图

### 1. 推荐
- 根据已存在的策略仓库，推荐高分标的
- 展示当前价格、使用策略、策略得分
- 根据策略的得分，命中推荐说明

## 技术架构

### Checking

#### 1. 是否需要服务器
- 应用功能较为简单，不打算外部使用，用户群体小
- 数据源为第三方平台，在客户端做服务和存储
- **结论：不需要服务器，纯客户端架构**

#### 2. 是否接入大模型
- 初期版本可以不接大模型
- 只需根据 API 的策略规则即可
- 甚至可以把规则同步到外部平台（如 ChatGPT）中生成算法，添加到规则策略管理中
- **结论：初期不接入，预留接口后续可接入**

#### 3. 如何同步
- 没有服务器的情况如何在多个设备中使用以及备份
- 多设备可以在局域网同步
- 技术栈使用 Flutter，可跨平台，两个终端之间实现备份
- **结论：局域网同步 + 本地备份/恢复**

## 技术栈（已有）

- **框架**: Flutter (跨平台 iOS/Android/macOS)
- **状态管理**: Riverpod
- **本地数据库**: Drift (SQLite)
- **网络请求**: Dio
- **路由**: GoRouter
- **本地存储**: SharedPreferences
